❓ Mode d'emploi — Tout savoir sur le site

Pourquoi ce site ?
Ce site permet à l'ENAM de collecter les fiches de renseignements des élèves en format numérique. Fini le papier, tout est centralisé et accessible en un clic. La direction crée les classes, envoie les liens aux élèves, et récupère les fiches remplies directement en Excel ou Word.

Qui peut voir quoi ?
- Les élèves → uniquement le formulaire (ils ne voient pas les autres fiches)
- L'administration → tout le tableau de bord (toutes les fiches, exports, journal)

Comment ça marche étape par étape :

1. Créer un formulaire
Allez dans l'onglet "Formulaires" → cliquez sur "+ Nouveau formulaire" → entrez le nom de la formation (ex: "Réseaux Informatique"), les années (ex: "2024-2027"), et le nombre de classes (ex: "3" pour 1ère, 2ème, 3ème Année). Validez.

2. Copier les liens des élèves
Une fois la promotion créée, chaque classe a un lien unique (ex: .../1ere-annee). Cliquez sur "Copier" à côté du lien. Envoyez ce lien aux élèves de cette classe par WhatsApp. Chaque classe a son propre lien — ne mélangez pas.

3. Les élèves remplissent le formulaire
L'élève clique sur le lien → voit sa classe affichée en haut → remplit tous les champs (Nom, Prénom, Téléphone, Adresse, Personne à contacter) → valide. Une fois soumis, c'est enregistré immédiatement.

4. Voir les fiches remplies (onglet Élèves)
Dans l'onglet "Élèves", toutes les fiches apparaissent dans un tableau. Utilisez le filtre en haut pour voir uniquement une classe spécifique. Vous pouvez modifier une fiche (bouton ✎) ou la retirer (bouton ✕).

5. Exporter en Word ou Excel
Dans l'onglet "Élèves", cliquez sur "Word" ou "Excel" pour télécharger la liste complète. Si vous avez sélectionné une classe dans le filtre, l'export ne contiendra que cette classe. Le fichier contient le logo ENAM, l'en-tête de la promotion, et toutes les fiches numérotées.

6. Consulter le Journal
L'onglet "Journal" enregistre chaque soumission (réussie ou refusée) avec la date, l'IP, le navigateur, l'appareil utilisé. Utile pour savoir qui a soumis quoi et depuis où.

7. Modifier le numéro WhatsApp
Dans l'onglet "Formulaires", vous pouvez changer le numéro WhatsApp qui apparaît sur la page d'accueil. Ce numéro permet aux nouveaux élèves de contacter la direction pour obtenir leur lien.

Conseils importants :
- Ne partagez jamais le lien /admin — il est réservé à la direction
- Envoyez chaque lien de classe uniquement aux élèves de cette classe
- Si un élève soumet deux fois, le système bloque le doublon automatiquement
- Les élèves retirés (✕) vont dans la section "Élèves retirés" en bas — vous pouvez les restaurer ou les supprimer définitivement
